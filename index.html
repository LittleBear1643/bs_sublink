BS Sublink

BS Sublink — лёгкий инструмент для создания ссылки для игры Brawl Stars с открытым исходным кодом. С помошью этого инструмента можно, например, подшутить над своим другом.

Создать ссылку,
перенаправляющую на
указанный URL-адрес
Создать ссылку,
выводящую определённое
сообщение
